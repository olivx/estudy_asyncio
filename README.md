#### install  project

- python 3.5+ is required
```bash
git clone git@github.com:olivx/estudy_asyncio.git asyncio_study
cd asyncio_study
python -m venv .venv 
source .venv/bin/activate 
pip install -r requirements.txt
```

####  run project study 
```python async_download_images.py```
